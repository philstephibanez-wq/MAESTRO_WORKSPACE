# P112D3 — ASAP Reference Book API Doc Pages

## Rôle

Enrichir le Reference Book ASAP avec des pages API/runtime couvrant les premières classes PHP 8 portées.

## Contrat

```text
NO DOC CONTRACT, NO PATCH
NO SILENT FALLBACK
NO MIXED RESPONSIBILITIES
```

## Pages documentées côté site

- Application Kernel
- SiteResolver
- Router
- TwigTemplateRenderer
- FSM Guard + ACL Guard
- Markdown documentation services

## Responsabilités

Le framework ASAP reste moteur mutualisable.

ASAP_REF_BOOK reste l’application documentaire qui rend ces informations via ASAP PHP 8 + Twig.
