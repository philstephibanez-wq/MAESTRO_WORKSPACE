# ASAP — Génération documentaire

## Rôle

Ce document définit la stratégie officielle de génération documentaire pour ASAP.

## Cible officielle

- Source API PHP : blocs PHPDoc compatibles phpDocumentor
- Source architecture : Markdown
- Diagrammes FSM : Mermaid
- Diagrammes ACL : Mermaid
- Sortie générée officielle : HTML
- Dossier de sortie HTML : DOC/reference/generated/html

## Contrat

NO DOC CONTRACT, NO PATCH.

NO HTML REFERENCE OUTPUT, NO DOC GENERATION VALIDATION.

## phpDocumentor

phpDocumentor est retenu pour extraire la référence PHP depuis les blocs PHPDoc.

## Mermaid

Mermaid est retenu pour les diagrammes FSM et ACL.

GraphViz devient une référence historique uniquement.

## HTML ASAP

La documentation générée doit pouvoir être consultée en HTML et rendue ou hébergée par ASAP.
