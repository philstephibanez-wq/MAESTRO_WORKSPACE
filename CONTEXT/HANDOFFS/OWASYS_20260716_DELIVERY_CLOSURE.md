## Generated application profiler contract

Every application generated by OWASYS receives the profiler automatically.

- mandatory; not selectable in the OWASYS UI;
- old request value `profiler: true` remains accepted for compatibility;
- `profiler: false` must fail with `OWASYS_PROFILER_MANDATORY`;
- generated files:
  - `config/profiler.json`
  - `application/default/helpers/GeneratedProfiler.php`
  - `www/asset/css/profiler.css`
  - `www/asset/js/profiler.js`
- profiler runtime is available only for `OPUS_ENV=dev`, `local`, or `development`;
- production returns no profiler instance even with `?profiler=1`;
- `/` and `?profiler=0` show no profiler;
- `?profiler=1` shows it only in an allowed development environment.

Current profiler implementation is a first functional implementation. Do not overclaim complete collector coverage yet.
