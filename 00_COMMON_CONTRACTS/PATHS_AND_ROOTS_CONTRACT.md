# Paths and Roots Contract

## Source roots

- H:\ASAP — mutualized PHP framework.
- H:\ASAP_REF_BOOK — RefBook documentation/control site.
- H:\MO_KB_DAEMON — MO_KB daemon application.
- H:\MO_KB_FRONT_ASAP — MO_KB PHP/ASAP frontend.
- H:\MAESTRO_WORKSPACE — workspace control center.

## Rule

A project source root is not a scratchpad.
A workspace folder is not a source repository unless explicitly defined as one.
